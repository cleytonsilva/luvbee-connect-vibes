# Luvbee Connect Vibes - Product Requirements Document

## Overview
Luvbee Connect Vibes é uma aplicação de conexão social baseada em localização que permite aos usuários descobrir locais noturnos e conectar com pessoas que compartilham interesses similares.

## Features

### 1. Autenticação
- Login e registro de usuários
- Gerenciamento de sessão
- Proteção de rotas

### 2. Onboarding
- Configuração inicial de preferências
- Coleta de informações do perfil

### 3. Vibe Local
- Descoberta de locais próximos
- Interface de swipe
- Geolocalização

### 4. Locations
- Lista de locais
- Filtros e busca
- Detalhes de locais

### 5. People Matching
- Descoberta de pessoas
- Sistema de matching
- Perfis de usuários

### 6. Messages
- Chat em tempo real
- Lista de conversas
- Mensagens não lidas

### 7. Profile
- Edição de perfil
- Configurações
- Estatísticas

## User Flows

### New User Flow
1. Acessa home page
2. Clica em "Criar Conta"
3. Preenche formulário de registro
4. Completa onboarding
5. Redirecionado para Vibe Local

### Existing User Flow
1. Acessa home page
2. Faz login
3. Navega pelas abas do dashboard
4. Interage com locais e pessoas
5. Envia mensagens

## Technical Requirements
- React 18+
- TypeScript
- Supabase backend
- Responsive design
- Accessible UI
